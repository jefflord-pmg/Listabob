## 3) Features & Functions

### 3.1 Core Functionality

#### **A. List Creation & Templates**

* Create lists from scratch
* Choose from built-in templates (Movies, TV shows, Video Game, Event itinerary, Asset manager, etc.)
* Import list structure and data from Excel tables
* Duplicate or reuse existing list structures

#### **B. Custom Columns & Data Types**

* Text, Numbers, Choice (dropdowns), Date/Time
* Lookup columns to relate data between lists
* People, Yes/No fields, Currency, Calculated fields
* Attachments and Images in list items

#### **C. Views & Visualizations**

* Grid view (spreadsheet-style)
* Calendar view
* Gallery (card) view with image previews
* Custom filtered and grouped views
* Sort and filter list contents

#### **D. Conditional Formatting & Rules**

* Highlight based on conditions (colors, formats)
* Smart rules to trigger alerts and basic automation
* Create if-this-then-that style logic directly in lists
